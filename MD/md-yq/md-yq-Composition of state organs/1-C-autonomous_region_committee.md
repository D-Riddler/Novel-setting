# 自治区委员会

自治区最高机构，由行政委员会，立法委员会，司法委员会组成。

## 行政委员会
### 主要职能
制定发展规划，进行社会管理，提供公共服务等职能。

## 立法委员会
### 主要职能
- 通过立法，审核并通过财政预算
- 对行政主席提名的人员进行表决批准
- 对高级官员提出弹劾并进行表决
- 举行听证会，对行政官员进行质询等

## 司法委员会
### 主要职能
- 提名高级法官和检察官
- 决定法官和检察官的任免
- 制订司法机关的经费预算
- 负责司法人员的资格考试
- 确定司法机构的设置等
